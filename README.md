# WhereECO iOS

## 개발환경

- Figma
- Xcode 14
- Swift
- UIKit, MapKit
- CocoaPods
- iOS 16.0

## 파일/폴더 안내
Air-Alarm-IOS/Air-Alarm
- /RestAPI : Rest API 소스
- /DTO : 데이터 객체 모음
- /Model : youtube cell 설정
- /LinkVC : Todo, youtube ViewController
- /MainVC : 로그인 ViewController
- /MapVC : 지도 ViewController
- /SignUpVC : 회원가입 ViewController
